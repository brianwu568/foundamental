# Foundamental LLM SEO - UI Frontend

A Rust/Rocket-based web interface for the Foundamental LLM SEO analytics platform.

## Features

- **Dashboard**: Overview of brand mentions, reliability scores, and key metrics
- **Brand Rankings**: Detailed view of brand rankings across LLM providers
- **Competitor Graph**: Interactive D3.js visualization of co-mention relationships
- **Hallucination Filter**: Risk analysis and source verification status
- **Provider Performance**: Compare LLM provider effectiveness

## Prerequisites

- Rust 1.70+ (install via [rustup](https://rustup.rs/))
- SQLite database (`llmseo.db`) generated by the Python backend

## Setup

1. **Install Rust** (if not already installed):
   ```bash
   curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
   ```

2. **Build the project**:
   ```bash
   cd ui
   cargo build --release
   ```

3. **Run the server**:
   ```bash
   cargo run --release
   ```

4. **Open in browser**:
   Navigate to `http://localhost:8000`

## Configuration

### Database Path

By default, the application looks for `../llmseo.db`. You can override this with an environment variable:

```bash
LLMSEO_DB_PATH=/path/to/your/llmseo.db cargo run
```

### Server Configuration

Edit `Rocket.toml` to modify server settings:

```toml
[default]
address = "127.0.0.1"
port = 8000
```

## API Endpoints

All API endpoints return JSON:

| Endpoint | Description |
|----------|-------------|
| `GET /api/stats` | Dashboard statistics |
| `GET /api/brands` | List of all brands |
| `GET /api/rankings` | Brand rankings data |
| `GET /api/providers` | Provider performance |
| `GET /api/graph?min_strength=0.0` | Competitor graph data |
| `GET /api/competitors/<brand>?top_n=5` | Top competitors for a brand |
| `GET /api/hallucinations` | Hallucination risk scores |
| `GET /api/quality` | Response quality metrics |
| `GET /api/sources/<mention_id>` | Sources for a mention |
| `GET /api/mentions?limit=50` | Recent mentions |
